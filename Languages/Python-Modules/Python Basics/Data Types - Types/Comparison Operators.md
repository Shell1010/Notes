# Comparison Operators

| Operator | Name                     | Example | Result |
| -------- | ------------------------ | ------- | ------ |
| ==       | Equal to                 | 5 == 5  | True   |
| !=       | Not equal to             | 5 != 5  | False  |
| >        | Greater than             | 5 > 3   | True   |
| >=       | Greater than or equal to | 5 >= 6  | False  |
| <        | Less than                | 5 < 6   | True   |
| <=       | Less than or equal to    | 6 <= 6  | True   |
|          |                          |         |        |
